# Brain Gym Academy App

Fresh start for a learning platform focused on:

- Soroban (traditional Japanese abacus)
- Phonics
- Parent-managed student progress

## Stack

- Frontend: Flutter
- Backend: Spring Boot
- Database: PostgreSQL

## Deployment direction

- Flutter Web on Firebase
- Spring Boot API on Render / Railway / similar host
- PostgreSQL on Neon / Supabase Postgres / Railway / Render Postgres

## Project structure

- `frontend/` - Flutter app
- `backend/` - Spring Boot API
- `docs/` - system design, API notes, database planning

## Current status

This project is intentionally starting from design-first scaffolding before feature build-out.
